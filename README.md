# vityarthi-project-healthcare-chatbot
MY PROJECT IS ON THE TOPIC HEALTHCARE CHATBOT WITH AN AIM TO PROVIDE BASIC INFORMATION ABOUT HEALTH RELATED QUESTIONS
THIS IS A RULE-BASED HEALTHCARE CHATBOT CREATED USING ONLY PYTHON CONCEPTS .THIS PROJECT SERVES AS A FOUNDATION IN THE HEALTHCARE CONTEXT.
IT OPERATES ON A SET OF PREDEFINED RULES AND KEYWORD MATCHING TO STIMULATE A BASIC CONVERSATION.IT DOES NOT USE ADVANCED MACHINE LEARNING MODELS INSTEAD ,IT RELIES ON A STRUCTURED DATABASE OF QUESTIONS AND ANSWERS.IT ENSURES PREDICTABLE AND CONTROLLED RESPONSE.
